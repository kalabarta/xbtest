Introduction:


Welcome to the X-ray Binary New Early Warning System (XB-NEWS) pipeline webpages. These webpages provide a graphical interface to the data processing and the output light curves. Full documentation for the XB-NEWS pipeline is available here.

The downloading of data by the XB-NEWS pipeline is driven by a target list that is split into active targets (classed as either regular or non-regular) and inactive targets. The pipeline searches for data corresponding to active targets in the Las Cumbres Observatory (LCO) archive on a regular basis, and it immediately downloads any new data that it finds. The pipeline does not search for data corresponding to inactive targets in the LCO archive. All downloaded data that exist locally are reduced by the XB-NEWS pipeline, regardless of whether the corresponding target is currently active or inactive, and the results are displayed on these webpages.
